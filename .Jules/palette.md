## 2026-02-02 - Scale Notes Copy Button
**Learning:** Adding a "Copy Notes" button to the ScaleNotes component provides a quick, useful way for users to extract the calculated scale notes. Providing immediate visual feedback (changing the icon to a checkmark and the text to "Copied!") for a short duration (2 seconds) confirms the action without requiring a separate toast notification. Using `navigator.clipboard.writeText` within a `try/catch` block ensures graceful failure if clipboard permissions are denied.
**Action:** When implementing copy-to-clipboard functionality, always include inline visual feedback that automatically reverts after a short delay to confirm the action to the user seamlessly.
